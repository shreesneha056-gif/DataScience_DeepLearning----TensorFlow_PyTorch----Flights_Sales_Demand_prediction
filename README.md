## 📌 Problem Statement
A framework comparison project that builds the same flight demand forecasting model in both PyTorch and TensorFlow to compare performance, API style, and ease of implementation.

## 🎯 Framework Comparison
| Framework | API Style | Training |
|-----------|-----------|----------|
| PyTorch | Dynamic graph, manual loop | Custom training loop |
| TensorFlow/Keras | High-level API | model.fit() |

## 🛠️ Tech Stack
- **Language:** Python
- **Deep Learning:** PyTorch, TensorFlow, Keras
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
